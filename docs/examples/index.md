# Code Examples

This section contains practical examples demonstrating InstrumentServer usage patterns.

## Available Examples

- **Basic Client** - Simple client connection and instrument control
- **Real-Time Monitoring** - Parameter monitoring and real-time updates
- **Measurement Sequences** - Multi-step measurement procedures
- **State Management** - Saving and restoring instrument states
- **Error Handling** - Robust error recovery patterns
- **Qt Integration** - Building Qt-based measurement applications

Examples are available in the repository under `test/test_async_requests/` and can be run as:

```bash
python test/test_async_requests/test_client.py
python test/test_async_requests/client_station_gui.py
```
